# 문자, 혹은 문장을 입력 받는 방법

```cpp
a  b  c  d  \n  e  f  \n  g  \n     //을 입력받는 다면..
```

## 1. cin>>
 - **화이트스페이스 전까지 입력** 받는다.
 - 처음 입력된 \n (개행)도 무시한다.
  - **화이트스페이스**
   - ' '(공백), \t, \n, \v, \f, \r
 - 위의 예에서는..
```cpp
(a  b  c  d)         // 처리
/n  e  f  /n  g  /n  // 남음
```

## 2. getline( cin , string)
 - \n 까지 입력받는다.
 - 받은 string에서 마지막 문자를 NULL로 안바꿈.
 - 위의 예에서는..
```cpp
(a  b  c  d)         // 처리
(/n)                 // 처리
e  f  /n  g  /n      // 남음
```
